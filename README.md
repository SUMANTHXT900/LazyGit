<h1>GitBuddy: A Simple Git GUI for Students, Developers, and End-Users</h1>

<p>GitBuddy is a simple and easy-to-use Git GUI designed for students, developers, and end-users. It provides a basic interface for performing common Git operations such as add, commit, push, and pull.</p>
<p>It is particularly useful for beginners who are new to Git and want a visual tool to help them learn and use Git commands.</p>

<h2>Features</h2>
<ul>
    <li>Simple interface for basic Git operations (add, commit, push, pull)</li>
    <li>Commit message entry with warning for empty messages</li>
    <li>Output display for Git command results</li>
    <li>Easy to use for beginners</li>
    <li>Lightweight and simple design</li>
</ul>

<h2>Usage</h2>
<ol>
    <li>Install Python and Tkinter.</li>
    <li>Run the <code>git GUI-V1.py</code> script.</li>
    <li>Enter your commit message in the text field.</li>
    <li>Click the buttons to perform Git operations.</li>
</ol>
